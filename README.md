# Dark-Forest

这个工具箱整理了多种实用的功能，如果想新增功能请在评论区中提出

## 🌲 黑暗森林工具箱 (Dark Forest Toolkit)

### 简介
为不会使用小众但实用工具的用户准备的集成工具箱，包含Windows内置工具和自定义实用功能。

### 功能列表

#### 🔧 系统工具
1. **系统信息查看** - 查看操作系统、硬件配置等详细信息
2. **进程管理** - 查看、搜索和管理系统进程
3. **磁盘空间分析** - 分析各分区使用情况
4. **环境变量查看** - 查看所有环境变量并支持搜索
5. **快速清理临时文件** - 一键清理系统临时文件

#### 🌐 网络工具
6. **网络工具集** - Ping测试、IP配置查看、端口扫描

#### 📁 文件工具
7. **文件批量重命名** - 按模式批量重命名文件

#### 🔐 加密与安全
8. **文本加密/解密** - Caesar密码、XOR加密
9. **密码生成器** - 生成高强度随机密码
10. **Base64编解码** - 快速进行Base64编码和解码
11. **哈希计算器** - 计算MD5、SHA1、SHA256等哈希值

#### 📄 数据处理
12. **JSON格式化** - 美化和格式化JSON数据
13. **时间戳转换** - 时间戳与日期时间互相转换

#### 📋 其他工具
14. **剪贴板工具** - 复制和读取剪贴板内容

### 使用方法

#### 直接运行
```bash
python dark_forest_toolkit.py
```

或者在Linux/Mac上：
```bash
./dark_forest_toolkit.py
```

#### 依赖安装（可选）
部分功能需要 `psutil` 库以获取更详细的系统信息：
```bash
pip install psutil
```

### 特点
- ✅ 跨平台支持（Windows/Linux/Mac）
- ✅ 纯Python实现，无需额外依赖（基础功能）
- ✅ 交互式菜单，简单易用
- ✅ 适合不熟悉命令行的用户
- ✅ 持续更新更多实用工具

### 注意事项
- 部分功能（如结束进程、清理文件）需要谨慎使用
- Windows某些功能可能需要管理员权限
- 建议在虚拟机或测试环境中先试用
