# 🌲 黑暗森林工具箱 - GUI 版本

## 简介

这是为小白用户特别设计的**图形化界面版本**，具有友好的操作界面和现代化的设计风格。

### ✨ 主要特点

- **🎨 现代化 UI 设计**：采用圆角卡片式设计，视觉舒适
- **🖱️ 点击即用**：无需输入命令，鼠标点击即可使用所有功能
- **📱 直观布局**：工具以网格形式排列，一目了然
- **🔍 实时反馈**：所有操作都有清晰的提示和结果展示
- **🛡️ 安全确认**：重要操作前会有确认对话框，防止误操作

## 🚀 快速开始

### 方法一：直接运行（推荐）

```bash
python dark_forest_toolkit_gui.py
```

### 方法二：双击运行（Windows）

在 Windows 系统上，可以直接双击 `dark_forest_toolkit_gui.py` 文件启动。

## 🎯 使用示例

### 生成密码

1. 点击主界面的「🔑 密码生成」卡片
2. 拖动滑块设置密码长度（8-64 位）
3. 勾选需要的字符类型（大写字母、小写字母、数字、特殊字符）
4. 点击「生成密码」按钮
5. 点击「复制到剪贴板」即可使用

### 批量重命名文件

1. 点击「📝 批量重命名」卡片
2. 点击「浏览...」选择要重命名的文件夹
3. 输入命名模式，如 `{name}_{num}.{ext}`
4. 点击「预览」查看效果
5. 确认无误后点击「执行重命名」

### JSON 格式化

1. 点击「📋 JSON 格式化」卡片
2. 在上方文本框粘贴 JSON 数据
3. 点击「格式化」按钮美化 JSON
4. 或点击「压缩」按钮最小化 JSON

## 💡 设计亮点

### 1. 圆角设计
- 所有按钮和卡片都采用圆角设计（半径 10-15px）
- 视觉效果柔和，符合现代 UI 趋势

### 2. 配色方案
- **主色调**：蓝色 (#4A90E2) - 专业、可靠
- **辅助色**：绿色 (#50C878) - 成功、安全
- **强调色**：红色 (#FF6B6B) - 警告、删除

### 3. 交互反馈
- 鼠标悬停时按钮颜色变亮
- 卡片边框高亮显示
- 操作完成有明确提示

### 4. 字体选择
- 中文：Microsoft YaHei UI（微软雅黑）
- 代码：Consolas（等宽字体）
- Emoji：Segoe UI Emoji

## ❓ 常见问题

### Q: 为什么窗口打不开？
A: 请确保已安装 Tkinter。在 Linux 上可能需要安装 `python3-tk` 包。

### Q: 某些功能无法使用？
A: 部分功能需要额外依赖，请参考「可选依赖」部分安装。

### Q: 如何退出程序？
A: 点击右上角的关闭按钮，或按 Alt+F4。
